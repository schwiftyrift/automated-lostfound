# automated-lostfound
This program is an automated Lost and Found system written in Python that utilizes object detection to label and store lost items in a database. This database is searchable, allowing people to find their lost items by a search. Item detection is implemented with a custom trained model based on the YOLO8 model, using OpenImages data. 

Libraries, tools, and resources used:
Ultralytics YOLO
OpenCV
PyTorch
OpenImages - https://storage.googleapis.com/openimages/web/index.html

Program by Jared Newton, Damjan Pavlovic, Rakesh Gupta for the 2025 Louisiana Tech Freshman Design Expo
